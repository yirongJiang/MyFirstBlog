## mongoose
### 介绍

项目入口是根目录下的 index.js

### 服务器启动
不启动文件监听 node index.js
启动文件监听 nodemon index.js


## 中间件
用来对mongodb 数据库进行操作的 node.js 的一个库
比如可以用来进行增删改查业务
```bash
pnpm add mongoose
```
解析 body
body-parser

解决跨域
cors
